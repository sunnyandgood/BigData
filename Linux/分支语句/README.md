## 分支语句
和其他语言相同，脚本的执行也是按照命令的顺序向下执行，前面的章节的例子都是按照命令的顺序逐条执行。
但是在实际的应用中，有时会需要逻辑判断，根据不同的条件选择不同的执行顺序。
